# Textbook Notes: Pearls of Functional Algorithm Design

Source PDF: `../textbook.pdf`

Bibliographic metadata extracted from the PDF front matter:

- Author: Richard Bird
- Title: Pearls of Functional Algorithm Design
- Publisher: Cambridge University Press
- First published: 2010
- ISBN-13: 978-0-521-51338-8 (hardback); 978-0-511-79588-6 (ebook)

These notes are bootstrap reference notes, not a replacement for the textbook. The book is organized as 30 short algorithm-design pearls. Most chapters are program calculations rather than theorem/definition sections, so the per-chapter files record problems, derivation anchors, key Haskell identifiers, and teaching-use notes instead of reproducing the chapter text.
